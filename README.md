## cbsp

cbsp is a header-only C++ library. It provides utility methods for archiving and extracting files. tar is complex and heavy but cbsp is simple and light, and can be easily added to your project.

### Comparation

#### archive

||cbsp|tar|
|--|--|--|
|1GB|0.642s|0.711s|

#### extract

||cbsp|tar|
|--|--|--|
|1GB|0.836s|0.862s|
